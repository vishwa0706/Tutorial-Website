## 📦 Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/vishwa0706/tutorial-website.git
cd tutorial-website
npm install
```

### Run locally

```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build for production

```bash
npm run build
```
